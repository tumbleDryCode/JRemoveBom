Quick Start
   
 
  If JremoveBOM is installed correctly you can just select the files you would like BOM´s removed from, right click over selected files and select the Send To -> JremoveBOM option.
JremoveBOM will launch and show you status of the files it is checking.
 
  Or, you can launch JremoveBOM from your computers Start -> All Programs -> JremoveBOM
You can than select the files you would like BOM's removed from by select the top File -> Open menu, and selecting the files you would like checked.
